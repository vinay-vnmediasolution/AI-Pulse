# AI Pulse: Advanced Intelligence Orchestration 🧠⚡

AI Pulse is a sophisticated, high-performance intelligence platform designed for AI engineers, researchers, and professional builders. It synthesizes real-time signals from across the frontier of artificial intelligence into an actionable, high-fidelity experience.

## 🚀 Vision
In an era of information explosion, AI Pulse acts as a **Neural Filter**. It doesn't just show news; it extracts logic, maps architectural dependencies, and provides a sandbox for conceptual testing.

---

## ✨ Key Features

### 📡 1. The Pulse Feed
A real-time, low-latency stream of the most critical AI developments. Powered by **Gemini 1.5 Pro with Google Search Grounding**, it filters through the noise to deliver high-signal intelligence.
- **Deep Extraction**: Every article includes an AI-generated analysis of "Why it Matters" and "Builder Impact".
- **Neural Sync**: Instantly bridge dossiers from the feed into the Laboratory.

### 📰 2. The Daily Specialist (Intelligence Briefing)
A curated, newspaper-style digital artifact generated every 24 hours. It targets specific high-value personas (like Vinay) to provide:
- **Strategic Signals**: Curated updates from Anthropic, OpenAI, and leading researchers.
- **Investment Alpha**: Business signals extracted from technical research.
- **Operational Tasks**: "Top 3 Actions" for the day, ensuring tactical focus.

### 🧪 3. The Neural Laboratory & Workbench
An interactive learning environment to demystify complex AI concepts.
- **The Workbench**: A dedicated console for each initialized module.
- **Playground**: Real-time inference testing on specific architectures (e.g., Workflow Engines, Agentic Chains).
- **Theory Docs**: Deep-dive whitepapers integrated into the UI for academic backing.

### 🔎 4. Intelligence Hub (Sources)
A high-integrity map of deep-dive dossiers and source materials, verified via Signal-C protocols.

## 🏗 Architecture

AI Pulse follows a **Modular Intelligence Architecture (MIA)**:

1.  **Ingestion Layer**: Utilizes Gemini's Function Calling and Search Grounding to scrape and synthesize high-density social and research streams.
2.  **Synthesis Engine**: A custom service layer that transforms raw JSON signals into structured `IntelligenceHubItem` or `NewsItem` objects, applying heuristic filtering for "Hype vs. Value".
3.  **The Workbench Pattern**: A state-driven orchestration where initialized "Nodes" create their own dedicated execution contexts, allowing for parallel sub-application state management within a single SPA.

## 🛠 Development & Setup

### Environment Variables
Create a `.env` file with the following:
```env
GEMINI_API_KEY=your_api_key_here
```

### Installation
```bash

npm run dev
```

### Build & Production
```bash
npm run build
npm start
```
